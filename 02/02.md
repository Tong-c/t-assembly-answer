### 檢測點2.1

##### 1.寫出每條匯編指令執行後相關寄存器中的值

| 指令            | 值           |
| --------------- | ------------ |
| mov   ax, 62627 | AX=**F4A3H** |
| mov   ah, 31H   | AX=**31A3H** |
| mov   al, 23H   | AX=**3123H** |
| add   ax, ax    | AX=**6246H** |
| mov   bx, 826CH | BX=**826CH** |
| mov   cx, ax    | CX=**6246H** |
| mov   ax, bx    | AX=**826CH** |
| add   ax, bx    | AX=**04D8H** |
| mov   al, bh    | AX=**0482H** |
| mov   ah, bl    | AX=**6C82H** |
| add   ah, ah    | AX=**D882H** |
| add   al, 6     | AX=**D888H** |
| add   al, al    | AX=**D810H** |
| mov   ax, cx    | AX=**6246H** |

##### 2.只能使用目前學過的匯編指令，最多使用4條指令，編程計算2的4次方

1. mov   ax, 2
2. mov   ax, ax
3. mov   ax, ax
4. mov   ax, ax

### 檢測點2.2

1. 給定段地址為0001H，僅通過變化偏移地址尋址，CPU的尋址範圍為 **0010H** 到 **1000FH** 
2. 有一數據存放在內存20000H單元中，現給定段地址為SA，若想用偏移地址尋到此單元，則SA應滿足的條件是：最小為 **1001H**， 最大為 **2000H**
